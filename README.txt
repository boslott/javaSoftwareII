Global Consulting Organization GUI-Based Scheduling Desktop Application

The purpose of this application is to maintain and interact with a data store of
customers, contacts, and appointment scheduling between them. The application can
create, read, update, and delete customers as well as create, read, update, and delete 
appointments.

To run the program:
1. Make sure the correct software is install on the machine:
  - Java
  - A Java IDE
2. Open the project files within an IDE
3. Build the project
4. Run the project


Custom Report - Customers By Country Division
    This report displays a list of all the divisions within the countries that 
the Global Consulting Organization operates out of alongside the country that 
each Division is contained.
    On the data side of the report, the total number of customers that live
within each of the respective Divisions is displayed.
